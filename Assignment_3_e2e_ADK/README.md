### A. Deep Research Agent for Lead Generation  
**Description:** Build a deep research agent that autonomously searches, summarizes, and extracts business leads.

### B. Advanced Tool Agent using Gemini CLI  
**Description:** Integrate Gemini CLI as a tool inside an ADK pipeline for enhanced automation and reasoning.  
**Reference:** [Combine ADK, Gemini CLI, and Cloud Run](https://medium.com/@derrickchwong/combine-adk-gemini-cli-and-cloud-run-c5dea5118853)

---

### C. MCP Tools-Based Agent for Bug Assistance  
**Description:** Build an agent that uses MCP (Model Context Protocol) tools to assist with debugging and development.  
**Reference:** [Tools: Make an agent from zero to assistant with ADK](https://cloud.google.com/blog/topics/developers-practitioners/tools-make-an-agent-from-zero-to-assistant-with-adk)

### D. Production-Quality Code Review Assistant  
**Description:** Create a code review assistant capable of analyzing, reviewing, and suggesting code improvements.  
**Reference:** [ADK Code Reviewer Assistant Codelab](https://codelabs.developers.google.com/adk-code-reviewer-assistant/instructions?hl=en#0)

### E. Production-Quality E-Commerce Agent  
**Description:** Build a production-ready e-commerce agent integrated with AlloyDB and ADK for product recommendation and support.  
**Reference:** [ADK Sports Agent with MCP & AlloyDB](https://codelabs.developers.google.com/codelabs/sports-agent-adk-mcp-alloydb?hl=en#8)
